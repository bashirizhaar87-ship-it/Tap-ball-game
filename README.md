# 25 * 4 = Tap Ball Game

Flappy Bird style HTML5 game. 1 file me pura game.

### 🎮 Kaise Khele
1.  Screen Tap karo = Game Start
2.  Tap karte raho = Ball Upar jayegi
3.  Pipe/Deewar se bacho

### 🚀 APK Kaise Banaye
`index.html` ko `AppMint`, `WebViewGold` ya `PWA Builder` me daal ke APK bana lo.
